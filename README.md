# Mine Sweeper

**C++ Qt 课程设计项目 Windows 经典扫雷小游戏**

## 项目简介

本项目为一个基于 Qt 制作的仿 Windows 经典扫雷小游戏，具有如下基本功能：

* 不同的难度等级
* 自定义游戏
* 记录和排行榜
* 作弊模式（在游戏中按下 Z 键）
